Estructura del patron State

  • Contexto: TrafficSimulator
  • Interfaz State: TrafficState
  • Estados concretos: FluentTrafficState, CongestedTrafficState, AccidentTrafficState, ClearedTrafficState
  • Cada estado decide como responder a las mismas acciones.
  • El mismo metodo produce distintos resultados segun el estado actual del objeto.

Cada estado no solo cambia el texto del sistema, tambien modifica semaforos, carriles, velocidades y decisiones del simulador.
